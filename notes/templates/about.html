All About Notes Sharing

“Notes Sharing Site” is a website created for students and teachers where they can upload their notes and can download as well. Notes that the user wants for a particular subject can be found in the website. This project is written in Python Django and sqlite. Firstly, the user need to sign up and then, they can log in to the system.

Notes Sharing Site is a open source you can Download and edit as per you need.
